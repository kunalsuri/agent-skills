# Lessons Learnt

Dated, self-contained design notes and decision records for the skills in this
repo. Each entry captures the *why* behind a choice — the positioning, the
trade-off, the alternative that was rejected — so it survives and isn't
re-litigated later. This is the same discipline the skills themselves teach:
write the reasoning down, commit it, let a future reader resume from it.

Append new entries; don't rewrite old ones. If a decision is reversed, add a new
dated entry that supersedes the old one and link back to it.
